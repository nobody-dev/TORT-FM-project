# TORT-FM Radio full page

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
gulp build
```

### After just start index.html in 'build' folder
```
build/index.html
```

### Technology stack:
```
- PUG (ex Jade)
- SCSS
- Javascript
- JQuery
- Self-made gulp
- FullPage.js library
```
